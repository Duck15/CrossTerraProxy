# CrossTerraProxy
## An Easy Way to Play Terraria Across Different Versions

> 🎯 No technical skills needed. Just run and play!

# How to Use:
1. Run the program — it will automatically create `config.json` for you.
2. Edit the file with your server info:

```json
{
  "proxy_port": "7778",                   // 🎮 Players connect to THIS port to join across versions
  "server_target": "127.0.0.1:7777",      // 🖥️ Your actual Terraria server address (IP:Port)
  "target_version_string": "Terraria318"  // 🏷️ The game version you want to "pretend" to be
}
```

3. Start your server, tell players to connect to `your-ip:7778`, and enjoy cross-version play! 🎉
