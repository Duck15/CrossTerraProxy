# CrossTerraProxy
## 一个傻瓜式的泰拉瑞亚网络版本验证替换器

# 使用方法：
运行程序 会生成`config.json`

```json
{
  "proxy_port": "7778",                   // 对于房主：玩家连到这个端口就可以跨版本了。  | 代理监听端口
  "server_target": "127.0.0.1:7777",      // 对于房主：这里需要填入房主服务器的IP。      | 代理后端服务器
  "target_version_string": "Terraria318"  // 对于房主：这里填入你的主机游戏版本。        |代理模拟版本
}
```
